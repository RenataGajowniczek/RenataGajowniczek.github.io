Overview

Overcome Autism is a visual identity for a concept foundation fighting against children’s autism. The project includes promotional materials, leaflet, patterns for bag and T-shirt, website’s layout and three posters.

Photoshop

Purpose and context

In order to expand my portfolio, I decided to work on a project that would be related to children. For some time I was searching for topics related to childhood illnesses and in the end, I decided to work on branding for children’s foundation fighting against autism.

I wanted to find a visual language that would be able to speak about a serious problem without painting it in a bad light. My main goal was to talk about the topics related to sickness in a playful and vigorous way typical for children in order to build a feeling of hope.

Objectives

1
Understanding the problem
Researching various children’s foundations in order to understand their main areas of activity.
2
Finding the visual theme
Researching topics related to solutions, seemingly unsolvable problems or dissimilarity.
3
Designing the brand
Concentrating on main elements for print and digital assets as well as goodies and takeaways.

Moodboard

Ribbons and tangled lines were the main visual elements that showcasing the seemingly unsolvable problems and dissimilarity. Bright colours and playful typography were supposed to connect the brand with children and approach the serious topic in the more easy-going manner.

Branding

The main visual language of Overcome Autism uses various shades of blue and rounded typography in order to show playfulness and vigour typical for children.

Learnings and takeaways

Overcome Autism is a project that uses tangled lines as a main element of the visual language. The biggest challenge was to expand those visual elements since the brand required a lot of printed and digital assets. I had to consider various layouts to create a space for both text and imagery. I think this project was a great exercise in creating a coherent brand.
